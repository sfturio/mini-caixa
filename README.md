# C# Fundamentals Store

Simple console application written in C# to practice programming fundamentals.

The project intentionally starts with **very simple and messy code**, and is gradually improved using:

- OOP
- Encapsulation
- Collections
- Interfaces
- Dependency Injection
- LINQ
- Better architecture

## Current features

- Product listing
- Simple sales system
- Basic discount rules
- Simple report

## Learning roadmap

Future improvements planned:
